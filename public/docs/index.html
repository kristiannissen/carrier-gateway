Logistics Gateway v1

A stateless, schema-first middleware for Scandinavian fulfillment.

View API Documentation

Unified Booking

Single JSON contract for multi-colli bookings across PostNord, GLS, DAO, and Bring. Integrate once, ship everywhere [2, 5, 6].

Trade Compliance

Automated validation for Non-EU shipments. Strict enforcement of HS Codes and Customs data based on DDP/DAP Incoterms [7, 8].

Reverse Logistics

Built-in support for labelless B2C returns via mobile QR code streaming and "Return of Goods" customs protection [6, 9].

Core API Endpoints

POST /api/v1/bookings Create shipment
GET /api/v1/tracking/:id Normalized tracking
GET /api/v1/service-points Carrier shop lookup
GET /api/v1/status System health
